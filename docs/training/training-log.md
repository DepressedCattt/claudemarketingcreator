# Animation Training Log

Chronological record of animation training sessions. Each entry captures what was taught, what was attempted, feedback received, corrections made, and rules extracted.

---

## How to Use This Log

After every training interaction:
1. Add a new session entry using the template below
2. Fill in all fields — especially "Before" and "After" values for parameter changes
3. If a generalizable rule emerges, note it in "Rule Extracted" and add it to `animation-fluidity.mdc`
4. After 3+ corrections on the same technique, create/update a technique card in `docs/training/techniques/`

---
